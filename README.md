# Python Scraping



## Setup
```
  python3 -m venv venv3
  
  pip install Scrapy

  scrapy --help

```
## create a template
```
  scrapy startproject ietf_scraper
  
  http://pythonscraping.com/linkedin/ietf.html

  scrapy genspider xxx exmaple.com
```

## run scrapy
```
   scrapy runspider test.py
```
